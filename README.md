# FoodieExpress - Vendor Management Portal

A comprehensive vendor dashboard built with React + TypeScript for store owners to manage their products, orders, and business operations on the FoodieExpress platform.

---

## Features

### Dashboard

· Real-time sales analytics and revenue tracking
· Order statistics (pending, confirmed, delivered)
· Popular products performance metrics
· Monthly earnings graphs

### Order Management

· View and manage incoming orders
· Update order status (confirmed → preparing → ready → delivered)
· Print invoices
· Order history with filters

### Product Management

· Add/Edit/Delete products with images
· Inventory management with stock alerts
· Bulk product upload via CSV
· Product categories and variants
· Discounts and special offers

### Store Operations

· Set operating hours and holidays
· Manage delivery zones and charges
· Toggle store open/close status
· Minimum order value configuration

### Analytics

· Daily, weekly, monthly sales reports
· Top selling products analysis
· Customer insights
· Download reports as PDF/Excel

### Reviews

· View and respond to customer reviews
· Rating analytics
· Report inappropriate reviews

### Profile Settings

· Store information management
· Logo and cover image upload
· Business details
· Password and security

---

## Tech Stack

· Frontend: React 18 + TypeScript
· Routing: React Router DOM
· Styling: Tailwind CSS
· Animations: Framer Motion
· Charts: Recharts
· Forms: Formik + Yup
· Notifications: React Hot Toast
· Icons: React Icons
· Build Tool: Vite

---

## Responsive Design

· Fully responsive on desktop, tablet, and mobile
· Collapsible sidebar for smaller screens
· Touch-friendly interface
